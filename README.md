# nodeJs-Typescript-boilerplate
Project boilerplate to build a nodeJs application using Typescript

## Usage
```javascript
  // install dependencies
    npm i
```
## Scripts included

```javascript
  // Run Dev
    npm run start:dev

  // Run Build
    npm run build

  // Run Prod
    npm run start

  // Run Prettier
    npm run prettier

  // Run Eslint
    npm run eslint

  // Run Jest ( test )
    npm run test

  // Run Jest ( test CI )
    npm run test:ci
```
